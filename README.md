## Introduce

This repository is Agent files for client connect to giip Service.

### Download giipAgent

```shell
git clone https://github.com/LowyShin/giipAgentLinux.git
```

### Configure client environment

```shell
vi giipAgent.cnf
```

Put your Secret Key and LSSN from giip web UI

Then save it.


### Register cron

```sh
sh giipcronreg.sh
```
