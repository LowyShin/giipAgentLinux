# CQE

CQE (Queue & Event) 관련 스크립트 및 문서

## 📄 파일 목록

### 실행 스크립트
- `giipCQE.sh` - CQE 메인 스크립트
- `giipCQECtrl.sh` - CQE 제어 스크립트

### 문서
- `CQE_TEST_GUIDE.md` - CQE 테스트 가이드

## 🚀 사용법

```bash
# CQE 실행
bash giipCQE.sh

# CQE 제어 (시작/중지/상태 확인)
bash giipCQECtrl.sh [start|stop|status]
```

## 📖 참고

더 자세한 테스트 방법은 `CQE_TEST_GUIDE.md`를 참고하세요.
